# httptest
一个模拟http请求的测试工具

## 开发环境
- Windows 10 1909
- PyCharm（Community Edition）
- Python 3.7
- 第三方库：
    1. requests：HTTP库，用于网络访问；
    2. beautifulsoup：网页解释库，提供lxml的支持；
    3. PyQt5：Python下的Qt库
    4. designer：绘制ui界面，利用qss美化
