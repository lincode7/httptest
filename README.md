# httptest
一个模拟http请求的测试工具  
1. 利用PyQt5绘制界面，包括请求构建部分（请请方法+url+请求头+请求参数/请求体）与响应展示部分；
2. 利用Requests模拟http请求，构建GET与POST的Http数据包；
3. 测试与优化。

## 开发环境
- Windows 10 1909
- PyCharm（Community Edition）
- Python 3.7
- 第三方库：
    1. requests：HTTP库，用于网络访问；
    2. beautifulsoup：网页解释库，提供lxml的支持；
    3. PyQt5：Python下的Qt库
    4. designer：绘制ui界面，利用qss美化
